Portfolio

allmarket (Github/ Deployed)

Allmarket is a platform designed to educate users on owning and trading cryptocurrency. Users can read simple explanations of cryptocurrency concepts, view a table of many active cryptocurrencies, and engage in mock trading of said currencies. This site functions primarily via jQuery, local browser storage, and Materialize javascript components.

Web Dev Quiz (Github/ Deployed)

This was a homework assignment to practice using javascript, jQuery, and local storage. It is a timed 11-question quiz where question order and answer order is randomized. There is a 'high score' table that records any completed quiz scores and saves them to the browser's local storage.

webM8

webM8 (Github/ Deployed)

WebM8 is the first truly full-stack application I've developed. It is a bookmark manager GUI application. Users can save bookmarks and organize them into collections, as well as add tags and a color to further help them categorize their bookmarks. This application utilizes an Express server and a MySQL database.
